How to use Gordon Voting System

  1. Create the election: Go to Admin → fill title, dates, toggles → Save election.
  2. Add candidates: Admin → add candidate name and optional tagline → Add candidate.
  3. Generate voter codes: Admin → set quantity → Generate → print or export CSV.
  4. Share codes: Distribute one code per voter securely.
  5. Voters cast votes: Voter → enter code → select candidate → submit.
  6. View results: Results → refresh as needed. Live results depend on the setting.
  7. Backup/restore: Admin → export/import JSON.

This is a client-only prototype. For production, use a secure server, audited cryptography, and independent verification.

Quick tips

  • Use Load demo data to try the system instantly.
  • Codes are single-use. If a code shows as used, it cannot vote again.
  • Importing JSON replaces the current election.
  • Reset clears all browser data for this prototype.
